Known Issues

  • Enabling Full Disk Access

    If Permanent Eraser is having problems erasing files in the Trash in macOS Catalina: Refer to the document Enabling Full Disk Access to resolve this issue.

  • Enabling the Erase Service Plug-In

    If the contextual plug-in doesn't work in macOS Mojave: Refer to the document Enabling the Erase Service Plug-In to resolve this issue.

  • Erasing Aliases

    An alias can only be deleted if it is in the Trash. Otherwise, trying to delete an alias will delete the original file, but not the alias.

  • Erasing Files

    Some files not owned by the current user may not be able to be immediately deleted. You will be prompted to enter in administrator credentials to authorize Permanent Eraser to erase this file. Files on a read-only disk, such as a CD-ROM, cannot be erased.

    Note: This feature is not available in the Mac App Store version of this app.
