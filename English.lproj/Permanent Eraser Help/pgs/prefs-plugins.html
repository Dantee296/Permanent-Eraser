Plug-Ins

To install the contextual plug-in for Permanent Eraser, select the Install Contextual Plug-in option in the Plug-Ins Preference pane.

Once the plug-in is installed, you can right-click on a file and select the option to erase the file.

Depending on the version of Mac OS X being used, the plug-in is installed in one of the following locations:

  • Mac OS X 10.5: /Users/[username]/Library/Contextual Menu Items/EraseCMI.plugin
  • Mac OS X 10.6 and later: /Users/[username]/Library/Services/Erase.workflow

Note: Plug-ins are not available in the Mac App Store of Permanent Eraser, so the Plug-Ins pane will not be available.
